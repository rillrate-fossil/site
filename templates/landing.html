RillRate

Dynamic tracing system that tends to be real-time.

Get started

Open-source MIT Licensed. GitHub v0.1.5

Low-latency

We use state of the art technologies to bring you a surprisingly fast system for monitoring your apps in real-time.

Zero-configuration

Your app will be automatically configured to provide only requested streams of metrics. Just point it to the server.

Live Dashboard

The library provides an embedded dashboard that can be used instantly as soon as you started your application.

Multiple Languages

We offer the library for many popular programming languages like: Rust, Python, Node.js. And many other languages will be supported soon.

Exporters

Easily transform your metrics into Prometheus or Graphite formats. Focus on metrics in your app, but not on exporting formats.

Teleport Agent

Get metrics and logs from existing apps with the simplicity of a single command line. Or connect it to Databases, Containers or other middleware.
